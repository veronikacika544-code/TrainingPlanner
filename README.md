## Описание
Training Planner - это GUI-приложение для планирования и отслеживания тренировок. Приложение позволяет добавлять, просматривать, фильтровать и удалять записи о тренировках, а также сохранять данные в формате JSON.

## Основные возможности
- ✅ Добавление тренировок с датой, типом и длительностью
- 📋 Табличное отображение всех тренировок
- 🔍 Фильтрация по типу тренировки и дате
- 💾 Сохранение и загрузка данных в/из JSON
- 📊 Отображение статистики тренировок
- ✅ Валидация вводимых данных
- 🎨 Современный интерфейс с цветовой схемой

## Требования
- Python 3.7 или выше
- Tkinter (обычно входит в стандартную установку Python)
